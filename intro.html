Introduction :

  • JS is a High Level Language (one can easily read).
  • Browser :- it is an application which provides environment to run js instructions from HLL to MLL.
  • JS code can run both inside and outside the browser. inside browser JS Engine and outside NODE JS engine.
  • Node :- It is the combination of C++ and v8(CHROME js engine)with bundle of methods and rules.
  • Node provides environment to run js outside the browser.This invention helps the js to gain its property in usage as a backend language.
  • We can run js inside and outside in the browser.

    • # Characteristics of Java Script

  • Purely object based.
  • Interpretated language.
  • Synchronus {It follows single thread architecture which has only one stack.}
  • Client side and server side.

    • # To execute java script on browser

  • We can use js instruction directly in the console directly provided in the browser.
  • We can execute js instruction by embedding it in the html file.
    1. Internal js
    2. External js

    1. Internal js

    Syntax :- < body> < script>console.log("hello")< /script> < /body>

  • With the help of script tag, we can embedd js instruction.

    • External js

    Syntax :- < body> < script src="filename.js"> < /script> < /body>

  • We can create a seperate file for js with a extention.js
  • Link the js file with html page using src attribute of script tag.
  • If the js file and html is in saame folder then file path is not supported.
