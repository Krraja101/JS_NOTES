Strings :

  • In js string can be represented using single quotes,double quotes or backtick.
  • NOTE : The start and end of a string must be done with the help of same quotes.
  • It a text contains single quotes than it can be represented using double quotes.
  • EX :- I'm a doctor in js :- "I'm a doctor"

    • Backtick :

  • A backtick can be multiline string.
  • It also gives a special way to concat the string with variables i.e ${variable name}

    • Some important methods of String :

    1. split() :
      • It is used to split the string into an array.
      • Syntax : split("")

      • EX:-
        let a="string"
        let d=a.split("")
        console.log(d)//["s","t","r","i","n","g"]
