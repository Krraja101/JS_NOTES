Token :

  • Smallest unit of any programming language consists of :
    1. Keywords :-
      • A word which understadable by the js engine is known as keyword.
      • NOTE : Every keyword must be in lowercase & is not used as identifiers .
      • EX :- for,let,const,while,null,switch do-while etc......

    2. Identifiers :-
      • Name given by the programmers to the components of js like variables ,functions class ele are known as identifiers.

        • Rules for identifiers :

      • An identifier can't start with number.
      • Except $ and _ no other special characters are allowed.
      • We can't use keywords as identifiers.

    3. Literals/Value :-
      • Data which is used is the js engine program is called as literals/values .

        • Types of values :

        1. Number
        2. String
        3. Boolean
        4. Null
        5. Undefined
        6. Object(non-premitive)
        7. Bigint
        8. Symbol
