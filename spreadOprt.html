Spread operator & Rest parameter :

Rest parameter :

  • Rest parameter is used to accept multiple values as an array of elements.
  • Generally it converts multiple values into an array.
  • Syntax : -> ...identifier

    • Uses of rest parameter :

  • Rest parameter can be used in function defination parameter list to accept multiple values .
  • It can also be used in array an object destructuring.

    • Using rest parameter in function :

  • The function can receive multiple arguments passed by caller and store them in a array.
  • EX :-

    • function check(a,...b) {
    log(a); // 10
    log(b); // [20,30]
    }
    check(10,20,30);

    NOTE -

  • Rest parameter is always used at the last.
  • EX :-

    • function check(...a,b) {
    log(a);
    log(b); // sysntax error: rest parameter must be last found
    }
    check(10,20,30);
