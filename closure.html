Closure :

  • The closure is a binding of parent function variables with the child function.
  • This enables a JS programmer to use parent function member inside child function.
  • EX:-

    • function person() {
    let a =21;
    function age(){
    return a;
    }
    return age ;
    }
    let x= person()
    log(x()); // 21

  • The above example age is a nested function of person in age function we didn't declare any local variable, but still we are able to use variable a which belongs to person function this is achived by closure.

    • NOTE-

  • The binding of child/inner/nesetd function with its lexical environment(parent function state) is known as closure.
  • Closure helps to achieve scope chain(lexical environment) from child function to parent function.
  • Closure preserves the state of parent function even after the execution of the parent function is completed.
  • A child function will have reference to the closure.
  • When ever parent function is called a new closure is created.
