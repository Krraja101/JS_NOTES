Function :

  • named block of instruction which is used to perform a specific task.
  • function get executed only when it is called.
  • the main advantage is we can achieve code reuseability.
  • Main purpose is code reuseability which means code once declared can be used anywhere.
  • We can create function in two ways
    1. Using function declaration statement(function statement)
      2.
      • function identifier(parameter,parameter.........){
        statement
        }

        • NOTE:-

      • Function is an object.
      • Name of a function is a variable which holds the reference of function object.
      • Creating a function using function statement supports function hoisting therefore we can also use a function before function declaration.
      • EX:-

        • log(test);
        finction test(){
        log('hi');
        }
        log("end");

      • When we try to log the function name the entire function will be printed.

        • PARAMETERS :-

      • The variable declared in the function defination/declaration is known as parameter.
      • The parameters have local scope,these can be used inside the function body .
      • parameters are used to hold the values which is passed during the calling a function.
      • EX:-

        • function sum(a,b){
        log(a+b)
        }

      • Here a & b are local variables to the sum function.

        • ARGUMENTS:-

      • The values passed in the function call statement is known as arguments.
        • \
      • An argument can be a literal,variable or an expression which results a value.
      • EX:-

        • sum(10,20) --> Here 10 & 20 are literals and used as arguments

        RETURN :

      • It is a keyword used as control transfer statement in a function.
      • Return will stop the execution of the function and transfers the control along with data to the caller.
      • EX:-

        • function Meter(centi){
        return (centi/100)
        }
        log(Meter(90))//0.9

        • Scope with respect to function : -

      • Any member declared inside a function will have local scope.

        • Local Scope :

      • The scope within the function block is known as local scope.
      • Any member with local scope can be used outside the function block.
      • A parameter of a function will have local scope.

    10. Callback function
      11.
      • Passing a function as a argument to another function is known as callback function.

        • Advantage :

      • Instead of creating a function with specific task ,functional programming helps programmer to generate function which can function a generic task by accepting a function as a argument.
      • EX :-

        • function operation(a,b,task){
        let res=task(a,b);
        return res;
        }
        let res1=operation(10,20,function add(m,n){return m+n; });
        log(res1);

      • In above example function add is a callback function.
